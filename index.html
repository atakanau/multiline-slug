Slugs and Domain Names
Creating slugs for domain names is a great way to turn common phrases in to searchable domain names. You can try multiple methods of using plain text to generate URL friendly domain names and permalinks. Slugifying is a common practice to get a short, unique, web friendly word from a phrase or name. Wordpress slug URLs are used within Wordpress to create unique permalink URLs for each post and page.
